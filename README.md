


# 腾讯会议自动入会管家
**基于 OpenCV 的定时入会辅助工具**



### 1. 核心功能描述

| 功能模块 | 说明 |
| :--- | :--- 
| **定时唤起** | 根据预设的星期和时间点，自动调用默认浏览器打开会议链接。 |
| **图像识别** | 利用 OpenCV 模板匹配算法，自动识别并点击网页按钮及浏览器协议弹窗。 |

### 2. 环境配置与启动

**安装依赖库**
```bash
pip install pyautogui opencv-python numpy customtkinter

```

**运行程序**

```bash
python TencentMeetingAuto.py

```


### 3. 运行前提：图像资源准备

程序依赖图像匹配进行操作。请根据你的实际显示环境，截取以下图片并存放在程序同级目录下：

| 文件名 | 截图内容说明 | 备注 |
| --- | --- | --- |
| `join_btn.png` | 网页中蓝色的“加入会议”按钮 | 建议尽可能仅截取按钮本身 |
| `open_btn.png` | 浏览器顶部的“打开腾讯会议”确认按钮 |  |

---

### 4. 部署发布 (Windows)

若需生成可独立运行的 `.exe` 文件，请使用 PyInstaller 进行打包。

**打包指令**

```bash
pyinstaller -F -w TencentMeetingAuto.py

```

> **注意：** 打包生成的 `.exe` 文件位于 `dist` 文件夹内。运行时，必须保证该文件与对应的 `.png` 截图文件处于同一文件夹路径。

---

### 5. 运行约束与风险提示

* **程序运行期间需保持系统处于唤醒状态。若电脑进入“睡眠”或“休眠”模式，定时任务将停止响应。**
* **截图时的系统 DPI 缩放倍率必须与程序运行时的倍率一致。缩放改变可能会导致识别失效。**
* **在执行点击动作时，程序需要接管鼠标控制权。请确保运行环境没有高优先级的安全软件拦截模拟点击操作。**
* **本工具仅负责自动拉起客户端。如果会议设置了等候室或需要主持人手动准入，仍需手动等待。**

### v1.0.2 更新日志

**新增功能**

* 新增实时倒计时显示：界面可直观显示距离下一次会议的剩余时间，且与后台监控互不影响。
* 新增防休眠机制：启动“自动入会”后，自动阻止 Windows 系统进入休眠模式，保障长时挂机。
* 新增环境自检：程序启动时会自动检测截图文件是否齐全，缺失时在日志中进行提示。

**性能与稳定性优化**

* 优化图像识别性能：增加图像缩放比例与识别区域缓存，大幅降低长期运行时的 CPU 占用。
* 解决内存溢出问题：限制日志框最大保留 80 行记录，防止挂机数天后程序卡顿。
* 解决多线程崩溃隐患：引入线程读写锁，修复在后台运行期间添加或删除日程可能导致的程序崩溃。
* 提升启停响应速度：优化后台调度逻辑。
* 提高入会成功率：优化唤起浏览器后的焦点抢占与最大化逻辑，减少因系统焦点丢失导致的点击失败。
* 修复界面布局问题：修复了缩小应用窗口时，底部控制区和按钮被挤压变形或遮挡的 UI 异常。
* 增强数据容错：完善了对本地数据文件读取和保存的异常捕获机制。
